# Design and Analysis of Algorithms

This repository contains Python implementations of Design and Analysis of Algorithms (DAA) laboratory experiments.

## Experiments

1. Interpolation Search
2. String Matching (Naive, Rabin-Karp, KMP)
3. Minimum Spanning Tree (Kruskal's & Prim's)
4. Dijkstra's Algorithm
5. Divide and Conquer – Min & Max
6. Matrix Chain Multiplication
7. N-Queens Problem
8. Travelling Salesman Problem (TSP)
9. Bin Packing Algorithm
10. Randomized Quick Sort

## Technologies Used

- Python 3
- Data Structures & Algorithms
